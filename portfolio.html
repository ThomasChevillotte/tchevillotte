PPM Image Processing.

School Project - 2022

This project focused on image processing, applying various filters to PPM images. The process began with understanding the PPM format and developing functions to read and create PPM files. Image processing features such as black and white conversion, binarization, rotation, and symmetry were implemented. Advanced techniques like convolution, zooming, blurring, erosion, and dilation were also explored. Filters could be applied via specific commands, with the program processing image data to generate the modified output.

Meteorological Data Processing.

School Project - 2022

This project involved processing meteorological data and visualizing it through graphs. A combination of a C program and a Bash script allowed users to filter data by date, geographical coordinates, or region. Over 2 million data points were processed to calculate averages, minimums, and maximums, and sort them based on user criteria. The processed data was then visualized through graphs, providing clear insights into variables such as temperatures, pressures, and wind speeds.

Image Morphing Application.

School Project - 2024

This project focused on developing an image morphing application as part of a first major Java project. A user interface was designed to separate visual, data, and calculation components. After researching the mathematical principles behind morphing, methods based on user-defined control points were implemented. The application generated frame-by-frame morph animations, combining them into a GIF file. This project highlighted the integration of mathematical concepts with software development.

Media Tracking Website.

School Project - 2024

ShowTime is a media tracking website designed for discovering and organizing movies and TV shows. Users can explore featured and popular content, search by genre, and access detailed information about films, series, and actors. They can also manage their personal lists by adding titles to "Favorites," "Watched," or "To Watch" Movies or TV Shows. Developed using Java and the Spring framework, the project incorporates a robust database populated with movie and series data fetched from an external API. Tailwind CSS was used to create a responsive and visually appealing interface, ensuring an intuitive user experience. This project also focused on user experience, with features like search filters and dynamic content update.
